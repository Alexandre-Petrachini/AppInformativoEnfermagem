Aplicativo Informativo Sobre Enfermagem

Este aplicativo tem o intuito de discutir sobre as práticas e métodos que serão utilizadas durante o curso de Enfermagem.

 

O Portugol consiste em uma extensão da linguagem com comandos, operadores, funções, funções recursivas, estruturas e construtores de estruturas complexas tais como stacks, queues e listas ligadas e listas duplamente ligadas.

Java é uma linguagem de programação e plataforma computacional muito utilizada em carros, computadores,  aplicações e sites, sendo portanto muito importante para o mundo atual.

O C# é uma linguagem de programação que surgiu com a finalidade de possibilitar a criação de soluções executáveis sobre a plataforma .NET Framework, com o intuito de flexibilizar o desenvolvimento de aplicativos.


O curso de Programação para Computadores (PC) envolve a utilização de todas as linguagens anteriormente vistas, e aspectos mais administrativos como modelo de testes e conexões.
